# Resume Website

## Overview
This is a simple personal resume website built to track progress in learning/using HTML and CSS.
Includes details about my background, technical skills, and links to my professional profiles and resume.

## Features
- Personal introduction section
- External links to GitHub, LinkedIn, and resume PDF
- Profile image section
- Organized technical skills grouped by category
- Clean, single-page layout

## Technologies Used
- HTML5
- CSS3

## Sections
- Header with navigation links
- About section with personal introduction
- Technical skills grouped into:
  - Programming languages
  - Computer architecture and assembly
  - Digital design and hardware
  - Frameworks and engines
  - Tools

## Purpose
This project was created to practice HTML and CSS fundamentals while building a personal resume 
website that can be expanded into a full portfolio in the future.
